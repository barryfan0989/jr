## ⭐ 功能使用說明

### 搜尋演唱會 🔍

1. 點擊搜尋欄
2. 輸入藝人名稱或地點（例如：「五月天」或「小巨蛋」）
3. 列表自動過濾

### 關注演唱會 ⭐

1. 在演唱會卡片上找到 **☆** 按鈕
2. 點擊變為 **★**（已關注）
3. 重新點擊可以 **取消關注**

### 設置提醒 🔔

1. 在演唱會卡片上找到 **🔕** 按鈕
2. 點擊變為 **🔔**（已設置提醒）
3. 重新點擊可以 **移除提醒**

### 查看演唱會詳情 📋

1. 點擊任何演唱會卡片
2. 彈出詳細信息 Modal，包含：
   - 🎤 演出藝人
   - 📅 演出時間
   - 📍 演出地點
   - 🌐 來源網站
   - 🔗 購票連結
   - 關注和提醒按鈕

### 查看我的關注 💖

1. 點擊「我的關注」標籤頁
2. 只顯示你已關注的演唱會
3. 搜尋仍然可用

## 🐛 遇到問題？

### 問題 1: 無法啟動
```
解決方案：
1. 檢查 Python 是否安裝: python --version
2. 檢查 Node.js 是否安裝: node --version
3. 重新運行 start.bat (Windows) 或 bash start.sh (Mac/Linux)
```

### 問題 2: 沒有演唱會數據
```
解決方案：
1. 爬蟲可能還在運行，請等待
2. 或手動運行: python concert_crawler.py --format json
3. 檢查終端是否有錯誤信息
```

### 問題 3: 登入後無法加載演唱會
```
解決方案：
1. 確保後端 API 在運行 (http://localhost:5000/api/health)
2. 清除瀏覽器緩存
3. 刷新頁面 (F5 或 Cmd+R)
```

### 問題 4: 密碼忘記了
```
解決方案（開發環境）：
1. 刪除 data/users.json
2. 重新啟動應用
3. 註冊新賬戶
```
